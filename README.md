## 구현 기능

+ Open Api를 이용하여 실제 어플리케이션과 유사한 형태로 개발
+ 큐싱 피해를 줄이기 위한 보안 검증 기능 추가
+ 큐싱 피해 발생 시 신속한 초동 대처 기능 추가
